# Mini-iPaaS Edge Monitoring Hub

**Platform:** Raspberry Pi 3 | Raspbian Bookworm
**Stack:** C (core engine) + Bash (wrapper) + cURL (Discord Webhook)
**Purpose:** Portfolio project for Fachinformatiker Systemintegration Ausbildung

## Project Structure
- src/       → C source code (log size reader engine)
- bin/       → Compiled executables
- scripts/   → Bash orchestration wrapper
- config/    → Webhook config (DO NOT COMMIT)
- docs/      → Architecture notes
- /var/log/ipass/ → Production runtime logs
